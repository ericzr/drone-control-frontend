# Fast Crud：详情展示规范（无人机模块）

## 原则

1. **全页详情**：需独立分享链接、打印、多区块深度阅读（事件详情、工单详情、设备详情页等）使用独立路由，如 `/event/detail?id=`。
2. **抽屉详情**：列表内快速预览、字段较少、以编辑为辅的交互保留 `Drawer`，并在需要时提供「在页面中查看」跳转全页。
3. **操作列**：`rowHandle` 中「详情」按钮与上述策略一致；同一实体避免抽屉与全页两套数据模型长期分叉。

## 现状与后续

- 各 `views/wemirr/drone-control/**/crud.tsx` 逐步对齐本规范。
- 接入真实 API 后，详情页统一通过 `id` 拉取接口，列表行内仅保留必要缓存字段。
