# 核心链路冒烟检查（手动）

登录后依次验证（开发环境）：

1. **机巢 → 监控**：`/device/dock` 对「驻机就绪」行点「一键起飞」→ 填参确认 → 进入 `/device/dock-monitor`，四分屏与返航确认正常。
2. **起飞 → 任务调度**：完成上一步后打开 `/dispatch/task`，列表顶部出现「一键起飞 · xxx」任务行（会话同步）。
3. **指挥中心终端**：`/situation-board` 切换场景、展开场景菜单、点终端卡片切换主视图与返回地图。
4. **事件 → 工单**：从事件列表进详情，关联工单链接可打开工单详情页。
5. **座舱键盘**：`/flight/cockpit` 打开喊话弹窗时，按 Q/W/E 不应触发飞行热键；关闭弹窗后热键恢复。

自动化：待引入 Playwright 后可将上述步骤脚本化。
